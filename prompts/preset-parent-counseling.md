# 프리셋: 학부모 상담 준비

## 프롬프트 템플릿

```
[학생 정보]
- 학교급: {{schoolLevel}}
- 학년: {{grade}}학년
- 유형: {{studentType}} (확신도 {{typeConfidence}}%)

[주요 검사 결과]
- 자기효능감: T={{tScores[1]}}
- 자기존중감: T={{tScores[0]}}
- 부모의사소통: T={{tScores[15]}}
- 부모학업지지: T={{tScores[16]}}
- 부모성적압력: T={{tScores[30]}}
- 부모공부부담: T={{tScores[31]}}
- 학업소진(고갈): T={{tScores[35]}}

[강점]
{{#each strengths}}
- {{factor}}: T={{score}}
{{/each}}

[보완점]
{{#each weaknesses}}
- {{factor}}: T={{score}}
{{/each}}

---

학부모 상담을 준비하고 있습니다. 다음 내용을 준비해주세요:

1. 학부모에게 전달할 긍정적 피드백 (강점 중심)
2. 가정에서 지원할 수 있는 구체적 방법 2-3가지
3. 학부모가 주의해야 할 점 (민감하게 전달할 내용)
4. 학부모의 예상 질문과 답변 준비
5. 상담 시 피해야 할 표현
```

## 예상 응답 형식

```
## 학부모 상담 가이드

### 1. 긍정적 피드백 (먼저 전달)
"자녀분은 {{strength}}에서 강점을 보이고 있습니다..."

### 2. 가정에서 지원 방법
1. **{{method1}}**: 구체적 설명...
2. **{{method2}}**: 구체적 설명...
3. **{{method3}}**: 구체적 설명...

### 3. 민감하게 전달할 내용
⚠️ 부모 관련 요인 점수가 높은 경우:
"가정에서의 기대가 자녀에게 동기부여가 되기도 하지만, 
때로는 부담으로 느껴질 수 있습니다..."

### 4. 예상 질문 & 답변
Q: "우리 아이가 왜 이런 결과가 나왔나요?"
A: ...

Q: "어떻게 해야 성적이 오르나요?"
A: ...

### 5. 피해야 할 표현
❌ "부모님이 너무 압박하셔서..."
✅ "가정에서의 지지 방식을 조금 조정하면..."
```

## 사용 시점
- 학부모 상담 전 준비
- AI 상담실에서 "학부모 상담" 프리셋 선택 시
