# 프리셋: 수업 중 개입 전략

```
[학생 정보]
- 학교급: {{schoolLevel}}
- 학년: {{grade}}학년
- 유형: {{studentType}}

[수업 관련 요인]
- 수업태도: T={{tScores[12]}}
- 수업부담: T={{tScores[27]}}
- 교사정서지지: T={{tScores[18]}}
- 교사성적압력: T={{tScores[33]}}
- 교사수업부담: T={{tScores[34]}}
- 자기효능감: T={{tScores[1]}}
- 활기: T={{tScores[19]}}
- 몰두: T={{tScores[20]}}

[메타인지 요인]
- 계획능력: T={{tScores[7]}}
- 점검능력: T={{tScores[8]}}
- 조절능력: T={{tScores[9]}}

[학업소진]
- 고갈: T={{tScores[35]}}
- 무능감: T={{tScores[36]}}

---

이 학생에게 수업 중 적용할 수 있는 개입 전략을 알려주세요:

1. 수업 참여도를 높이기 위한 전략 2가지
2. 이 학생에게 효과적인 피드백 방법
3. 모둠 활동 시 역할 배치 제안
4. 수업 중 주의해야 할 학생 반응 신호
5. 일과 중 1분 개입 아이디어 (쉽게 실천 가능한 것)
```

## 사용 시점
- 수업 계획 시
