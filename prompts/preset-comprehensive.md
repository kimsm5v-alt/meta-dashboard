# 프리셋: 종합 분석

## 프롬프트 템플릿

```
[학생 정보]
- 학교급: {{schoolLevel}}
- 학년: {{grade}}학년
- 유형: {{studentType}} (확신도 {{typeConfidence}}%)

[T점수 - 11개 중분류]
{{#each subCategoryScores}}
- {{name}}: T={{score}}
{{/each}}

[강점 요인]
{{#each strengths}}
- {{factor}}: T={{score}}
{{/each}}

[취약 요인]
{{#each weaknesses}}
- {{factor}}: T={{score}}
{{/each}}

---

이 학생의 META 검사 결과를 종합적으로 분석해주세요.

1. 전반적인 학습심리정서 상태 요약
2. 가장 주목해야 할 강점 2가지
3. 우선적으로 지원이 필요한 영역 2가지
4. 이 학생 유형({{studentType}})의 일반적 특성과 이 학생만의 특이점
5. 담임교사로서 가장 먼저 시도해볼 수 있는 개입 1가지
```

## 예상 응답 형식

```
## 종합 분석 결과

### 1. 전반적 상태
이 학생은 {{studentType}}으로 분류되었으며, ...

### 2. 주목할 강점
- **{{strength1}}**: ...
- **{{strength2}}**: ...

### 3. 지원 필요 영역
- **{{weakness1}}**: ...
- **{{weakness2}}**: ...

### 4. 유형 특성 vs 개인 특이점
[유형 일반 특성]
...

[이 학생의 특이점]
...

### 5. 추천 첫 개입
...
```

## 사용 시점
- 학생 대시보드 첫 진입 시
- "전체 분석 보기" 버튼 클릭 시
- AI 상담실에서 "종합 분석" 프리셋 선택 시
