# 프리셋: 맞춤 학습 경로 (지식그래프 기반)

## 프롬프트 템플릿

```
[학생 정보]
- 학교급: {{schoolLevel}}
- 학년: {{grade}}학년
- 유형: {{studentType}} (확신도 {{typeConfidence}}%)

[유형별 추천 개입 경로]
{{#each interventions}}
경로 {{@index}}: {{x}} {{#if z}}× {{z}}{{/if}} → {{y}}
효과 유형: {{effectType}}
해석: {{interpretation}}
{{/each}}

[현재 T점수]
{{#each relevantScores}}
- {{factor}}: T={{score}}
{{/each}}

[강점]
{{#each strengths}}
- {{factor}}: T={{score}}
{{/each}}

[취약점]
{{#each weaknesses}}
- {{factor}}: T={{score}}
{{/each}}

---

이 학생의 학업성취도/학습태도 향상을 위한 맞춤 개입 경로를 제안해주세요:

1. 가장 효과적인 개입 경로 1개 선택 및 이유
2. 해당 경로의 단계별 실행 계획 (4주 기준)
3. 경로 실행 시 활용할 수 있는 구체적 활동/자료
4. 효과 측정 방법 (어떻게 변화를 확인할 것인가)
5. 경로 실행이 어려울 경우 대안 경로
```

## 예상 응답 형식

```
## 맞춤 학습 경로 제안

### 1. 추천 개입 경로
**{{x}} → {{mediator}} → {{y}}**

선택 이유:
- 현재 {{x}}의 T점수가 {{xScore}}로 ...
- {{mediator}}를 통해 ...
- 최종적으로 {{y}} 향상 기대

### 2. 4주 실행 계획

| 주차 | 목표 | 활동 | 체크포인트 |
|------|------|------|-----------|
| 1주 | {{goal1}} | {{activity1}} | {{check1}} |
| 2주 | {{goal2}} | {{activity2}} | {{check2}} |
| 3주 | {{goal3}} | {{activity3}} | {{check3}} |
| 4주 | {{goal4}} | {{activity4}} | {{check4}} |

### 3. 활용 가능한 활동/자료
1. **{{resource1}}**
   - 활용 방법: ...
   - 다운로드/참고: ...

2. **{{resource2}}**
   - 활용 방법: ...

### 4. 효과 측정 방법
- 단기 지표 (2주): {{shortTermMetric}}
- 중기 지표 (4주): {{midTermMetric}}
- 관찰 포인트: {{observationPoints}}

### 5. 대안 경로
만약 위 경로가 어려울 경우:
**{{alternativePath}}**
이유: ...
```

## 지식그래프 효과 유형 설명

| 효과 유형 | 의미 | 개입 방향 |
|----------|------|----------|
| 긍정강화 | 두 요인 결합 시 시너지 | 동시 강화 |
| 긍정완충 | 과도하면 효과 감소 | 순차적 개입 |
| 부정강화 | 부정 요인 중첩 시 악화 | 하나씩 해결 |
| 촉진 | Z가 높을수록 X 효과 증가 | Z 먼저 강화 |
| 억제 | Z가 높을수록 X 악영향 감소 | Z로 방어막 |

## 사용 시점
- 학생 맞춤 지도 계획 수립 시
- 장기 개입 전략 설계 시
- AI 상담실에서 "학습 경로" 프리셋 선택 시
