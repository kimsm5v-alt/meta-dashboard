# 프리셋: 생활기록부 문구 생성

## ⚠️ 중요 규칙

1. **학생 이름 절대 포함 금지** → 교사가 수동으로 삽입
2. **주어 생략 또는 "학생은"으로 시작**
3. **글자 수 제한**: 기본 200자, 조절 가능 (100~500자)
4. **객관적 사실 + 긍정적 성장 가능성 중심**

---

## 프롬프트 템플릿

```
[학생 정보]
- 학교급: {{schoolLevel}}
- 학년: {{grade}}학년
- 유형: {{studentType}}

[주요 강점]
{{#each strengths}}
- {{factor}}: T={{score}}
{{/each}}

[성장 필요 영역]
{{#each weaknesses}}
- {{factor}}: T={{score}}
{{/each}}

[유형 특성]
{{typeDescription}}

[차수 변화] (2차 검사 완료 시)
- 1차 유형: {{round1Type}}
- 2차 유형: {{round2Type}}
- 주요 변화: {{significantChanges}}

---

생활기록부 "행동특성 및 종합의견"에 들어갈 문구를 작성해주세요.

요구사항:
1. 글자 수: {{targetLength}}자 내외
2. 학생 이름 절대 포함하지 않기 (주어는 생략하거나 "학생은"으로)
3. 강점을 먼저 언급하고, 성장 영역은 긍정적 표현으로
4. 구체적인 행동/태도 묘사 포함
5. 미래 성장 가능성/기대 문장으로 마무리
```

## 예상 응답 형식 (200자 기준)

```
학업에 대한 의지가 있으며 수업 시간에 성실하게 참여하는 모습을 보임. 
특히 {{strength1}}에서 강점을 보이며, 또래와의 관계에서도 긍정적인 
모습을 나타냄. {{weakness1}}의 경우 꾸준한 연습이 필요하나, 
포기하지 않고 노력하는 자세가 돋보임. 
앞으로 자신감을 가지고 도전한다면 더욱 성장할 것으로 기대됨.
```

## 글자 수별 템플릿

### 100자 (간략)
```
{{strength}} 영역에서 강점을 보이며 수업에 성실히 참여함. 
{{weakness}} 보완을 위한 노력이 필요하나 성장 가능성이 높음.
```

### 200자 (기본)
```
(위 예시 참조)
```

### 300자 (상세)
```
학습 활동에 적극적으로 참여하며, 특히 {{strength1}}과 {{strength2}}에서 
또래 대비 우수한 모습을 보임. 수업 시간에는 {{classroomBehavior}} 하는 
태도가 인상적임. {{weakness1}}의 경우 현재 다소 어려움을 겪고 있으나, 
{{improvement}} 하려는 의지가 엿보임. {{supportStrategy}} 등의 지원을 
통해 해당 영역의 성장이 기대됨. 전반적으로 {{overallComment}} 하며, 
지속적인 격려와 관심 속에서 더욱 발전할 것으로 기대됨.
```

### 500자 (종합)
```
(강점, 학습태도, 대인관계, 성장영역, 지원방안, 기대 등 상세 기술)
```

## 유형별 키워드 가이드

### 자원소진형
- 강점 표현: "어려움 속에서도 포기하지 않는", "꾸준히 노력하는"
- 성장 표현: "자신감 회복을 통해", "작은 성공 경험을 쌓아가며"

### 안전균형형
- 강점 표현: "안정적인 학습 태도", "균형 잡힌 모습"
- 성장 표현: "계획적인 학습 습관을 통해", "자기 점검 능력을 키워"

### 몰입자원풍부형
- 강점 표현: "학습에 대한 높은 열의", "자기주도적인 학습 태도"
- 성장 표현: "시험 전략을 보완하여", "도전적인 과제를 통해"

---

## 사용 시점
- 학기말 생활기록부 작성 시
- "생활기록부 문구 생성" 버튼 클릭 시
- 대량 생성 (30명 일괄) 기능 사용 시

## 주의사항
- 생성된 문구는 반드시 교사가 검토 후 수정
- 학생 이름은 교사가 직접 삽입
- 동일 학급 내 문구 중복 주의
