# META 학습종합검사 개요

## 1. 검사 소개

META 학습종합검사는 **한국심리학회 공인 인증(KPA 2201)**을 받은 신뢰도와 타당도가 검증된 학습 진단 도구입니다.

### 1.1 검사 목적
- 학생들의 학습 과정에서 겪는 어려움을 조기에 발견
- 효과적인 학습 방향 설정 지원
- 학업성취에 영향을 미치는 심리·정서적 요인 종합 진단

### 1.2 검사 대상
- 초등학생
- 중학생
- 고등학생

### 1.3 검사 소요 시간
- 약 30분 내외 (온라인 검사)

---

## 2. 이론적 배경

### 2.1 JD-R (Job Demands-Resources) 모형

검사는 **Demerouti 등(2001)**이 제안한 JD-R 모형을 학업 맥락에 적용하여 설계되었습니다.

```
┌─────────────────────────────────────────────────────────────┐
│                    JD-R 모형 적용                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   학업 요구 (Demands)          학습 자원 (Resources)         │
│   ├── 부모의 기대              ├── 학습 전략                 │
│   ├── 성적 부담                ├── 교사의 지지               │
│   └── 과도한 학습량            └── 긍정적 자아               │
│          │                            │                     │
│          ▼                            ▼                     │
│     학업 소진 ←───────────────→ 학업 열의                   │
│          │                            │                     │
│          └──────────┬─────────────────┘                     │
│                     ▼                                       │
│               학업 성취도                                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 핵심 원리
- **학업 요구**가 지나치면 → 학업 소진 발생 가능성 ↑
- **학습 자원**이 충분하면 → 학업 열의 ↑ → 성취 ↑

> 참고문헌: Salmela‐Aro & Upadyaya (2014), 문경용 외 (2018)

## 3. 검사의 특장점

### 3.1 신뢰도 높은 공인 검사
- 한국심리학회 인증
- 온라인으로 간편하게 검사 가능
- 교사 업무 부담 최소화

### 3.2 학기 초 학생 파악 용이
측정 요인:
- 긍정적 자아
- 대인관계능력
- 메타인지
- 학습기술
- 지지적 관계
- 학업열의
- 성장력
- 학습 걸림돌
- 부정적 공부마음
- 학업 스트레스

### 3.3 객관적 평가 자료 제공
- 학생 생활기록부 작성 시 근거 자료로 활용
- T점수와 백분위로 객관적 비교 가능

### 3.4 데이터 분석 기능
- 학급 전체 데이터 일괄 분석
- 반 평균 분석을 통한 학급 경향 파악
- 개별 맞춤 지도 가능

---

## 4. 활용 방안

### 4.1 교사용
| 활용 목적 | 내용 |
|-----------|------|
| 학급 운영 | 학기 초 학생 특성 파악 및 맞춤형 학습 지도 계획 수립 |
| 학생 평가 | 객관적인 데이터 기반 학생 생활기록부 작성 |
| 집단 지도 | 학급 공통 문제 파악 및 집단 지도 전략 수립 |

### 4.2 학생/학부모용
- 학습 강점과 약점 객관적 진단
- 보완 필요 영역 파악
- 구체적인 학습 방향 제시

---

## 5. 검사 결과 해석 체계

검사 결과는 **심리검사 기반 학습코칭(Assessment Based Coaching)** 체계로 구성됩니다.

```
┌────────────────┐    ┌────────────────┐    ┌────────────────┐
│   Assessment   │ → │    Analysis    │ → │    Coaching    │
│   학습현황 파악 │    │  학습진단/해석  │    │   학습성장 코칭 │
└────────────────┘    └────────────────┘    └────────────────┘
```

### 해석 5단계 (STEP)
1. **STEP 1**: 해석 가이드 - 점수 체계 이해
2. **STEP 2**: 자아강점 - 긍정적 자아, 대인관계능력
3. **STEP 3**: 학습 디딤돌 & 긍정적 공부마음
4. **STEP 4**: 학습 걸림돌 & 부정적 공부마음
5. **STEP 5**: 종합 해석 - 실천 방안 제시
