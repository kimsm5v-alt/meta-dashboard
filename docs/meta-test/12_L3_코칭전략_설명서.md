# L3 코칭 전략 — 기술 설명서

> L3(학생 대시보드)의 "코칭 전략" 모달은 학생의 LPA 유형과 38개 T점수를 기반으로
> **개인화된 코칭 경로**를 관련도 순으로 제시하는 기능입니다.

---

## 1. 전체 구조

```
[학생의 38개 T점수] + [LPA 유형]
        ↓
  interventionRanker.ts ← lpaProfiles.ts (유형별 경로 풀)
        ↓                  knowledgeGraph.ts (매개경로 + 조절효과)
  관련도 점수 산출 → 정렬
        ↓
  CoachingStrategy.tsx (Top 5 표시 + 더보기)
```

| 레이어 | 파일 | 역할 |
|--------|------|------|
| 데이터 | `knowledgeGraph.ts` | 노드 29개 + 엣지 38개 (매개경로 13 + 조절효과 25) |
| 데이터 | `lpaProfiles.ts` | 유형별 intervention 총 57개 (랭킹 대상) |
| 로직 | `interventionRanker.ts` | 관련도 점수 산출 + 정렬 |
| UI | `CoachingStrategy.tsx` | 모달 표시, Top 5 + 더보기 |

## 3. 엣지(Edge) 관계 유형

노드 간 관계를 정의하는 **5가지 RelationType**이 있습니다.

| relation | 한글명 | 방향 | 설명 |
|----------|--------|------|------|
| `needs_mediation` | 매개 필요 | 학생집단 → 매개변수 | 이 유형은 해당 매개변수를 경유해야 결과 달성 가능 |
| `mediates_to` | 매개 경로 | 매개변수 → 결과변수 | Z가 X와 Y 사이를 매개 (SEM 간접효과, β 검증) |
| `requires_intervention` | 개입 필요 | 학생집단 → 개입전략 | 이 유형에 해당 개입전략이 필요 |
| `has_strength` | 강점 보유 | 학생집단 → 매개변수 | 이 유형이 해당 매개변수에서 강점을 가짐 |
| `moderated_effect` | 조절 효과 | 학생집단 → 결과변수 | X×Z 상호작용이 Y에 영향 (상호작용 β 검증) |

### 3.1 매개경로 엣지 vs 조절효과 엣지

| 구분 | 매개경로 (`mediates_to`) | 조절효과 (`moderated_effect`) |
|------|--------------------------|------------------------------|
| **구조** | X → Z → Y (순차적 경유) | X × Z → Y (상호작용) |
| **통계 모델** | SEM 간접효과 (a×b) | SEM 상호작용항 (X·Z) |
| **속성** | `mediationType` (완전/부분매개), `간접효과_β`, `간접효과_CI` | `moderatorType` (긍정강화/긍정완충/촉진/억제), `상호작용_β` |
| **KG 엣지 수** | 13개 (모델 단위 그룹화) | 25개 (X×Z별 개별) |
| **해석** | "X가 Z를 거쳐야 Y에 도달" | "Z 조건에 따라 X→Y 효과가 달라짐" |

### 3.2 매개경로 엣지 통계 구조 (`MediationStats`)

```
{
  경로: 'X → Z → Y',
  a경로_β: number,       // X → Z 경로 계수
  a경로_p: string,
  b경로_β: number,       // Z → Y 경로 계수
  b경로_p: string,
  간접효과_β: number,    // a × b (핵심 지표)
  간접효과_CI: [lo, hi], // 95% 신뢰구간
  직접효과_β?: number,   // X → Y 직접 (완전매개 시 비유의)
  해석: string
}
```

### 3.3 조절효과 엣지 통계 구조 (`ModerationStats`)

```
{
  독립변수: string,       // X
  조절변수: string,       // Z
  상호작용_β: number,    // X × Z 상호작용 계수
  상호작용_p: string,
  해석: string,
  개입전략: string        // 조절효과 기반 권장 행동
}
```

## 4. 코칭 경로(Intervention)란?

하나의 코칭 경로는 **"어떤 요인(X)에 개입하면, 어떤 메커니즘(Z)을 거쳐, 어떤 결과(Y)가 나오는가"**를 정의합니다.

```
Intervention {
  x: string          // 독립변수 (개입 대상 요인)
  z: string | null    // 매개변수 또는 조절변수
  y: string           // 결과변수 (학업성취도 또는 성적만족도)
  effectType: string  // 효과 유형 (완전매개, 긍정완충, 촉진 등)
  beta?: number       // 통계적 효과크기 |β| (양수로 저장)
  source?: string     // 데이터 출처 (KG / KG_INTERVENTION / INFERRED)
  interpretation: string   // 해석 문장 (교사가 읽을 수 있는 형태)
  strategies: string[]     // 구체적 실행 전략 (3개씩)
}
```

### 4.1 효과 유형 (EffectType) 분류

각 경로는 **매개경로**, **조절효과**, **직접효과** 중 하나로 분류됩니다.

| 분류 | 효과 유형 | 기호 | 의미 | 예시 |
|:---:|-----------|------|------|------|
| **매개경로** | 완전매개 | X → Z → Y | X가 Z를 경유해야만 Y에 영향 | 자기효능감 → 성장마인드셋 → 학업성취도 |
| **조절효과** | 긍정강화 | X × Z ↑↑ → Y↑ | X와 Z가 결합하면 시너지 | 타인정서인식 × 활기 → 성적만족도 |
| **조절효과** | 긍정완충 | X↓ × Z↑ → Y 유지 | Z가 높으면 X 부족의 부정적 영향 완충 | 점검능력↓ × 교사정서지지↑ → 성적만족도 유지 |
| **조절효과** | 촉진 | X × Z → Y 증폭 | Z 수준에 따라 X의 효과가 증폭 | 수업부담 × 교사정서지지 → 학업성취도 |
| **직접효과** | 직접효과 | X → Y | Z 없이 X가 Y에 직접 영향 | 시험준비 → 학업성취도 |

### 4.2 데이터 출처(Source) 3-tier

| 출처 | UI 배지 | β 유무 | 신뢰도 | 설명 |
|------|---------|:---:|:---:|------|
| **KG** | `β=0.XXX` (indigo) | O | 최고 | 지식그래프의 SEM 분석에서 통계적으로 검증된 경로 |
| **KG_INTERVENTION** | `β 없음` (amber) | X | 중간 | 지식그래프에 개입전략 노드가 존재하나, β 미검증 |
| **INFERRED** | `AI 추론` (purple) | X | 참고 | 유형 특성 기반 AI 추론 경로. 추후 검증 필요 |

## 5. 관련도(Relevance Score)란?

**관련도**는 특정 코칭 경로가 **이 학생에게 얼마나 시급하고 적합한지**를 0~100 사이 점수로 나타낸 것입니다.
같은 유형의 학생이라도 38개 T점수 프로필이 다르므로, 관련도 점수와 경로 순위가 달라집니다.

### 5.1 산출 공식

```
관련도 = needScore × 0.5 + deviationScore × 0.3 + betaBoost × 0.2
```

| 요소 | 가중치 | 의미 | 산출 방법 |
|------|:---:|------|------|
| **needScore** | 50% | 개입 필요도 | 관련 요인(X, Z)의 T점수가 이상적 방향에서 벗어난 정도. 정적 요인은 50점 미만일수록, 부적 요인은 50점 초과일수록 높아짐 |
| **deviationScore** | 30% | 유형 내 취약도 | 같은 유형의 평균 T점수 대비 이 학생이 나쁜 방향으로 벗어난 정도. 나쁜 방향 편차에 1.5배, 나은 방향에 0.5배 가중 |
| **betaBoost** | 20% | 검증된 효과크기 | 통계적 β 계수가 클수록 높아짐. β=0.479이면 95.8점, β=0.045이면 9.0점 |

### 5.2 산출 예시

> **초등 안전균형형 학생** (점검능력 T=35, 조절능력 T=38, 유형평균 점검 T=42, 유형평균 조절 T=45)
>
> 경로: `점검능력 → 조절능력 → 학업성취도` (β=0.117)
>
> - needScore = ((50-35) + (50-38)) / 2 × (100/30) = **45**
> - deviationScore = ((42-35)×1.5 + (45-38)×1.5) / 2 × (100/15) = **70**
> - betaBoost = 0.117 × 200 = **23**
> - **관련도 = 45×0.5 + 70×0.3 + 23×0.2 = 48**

### 5.3 UI 표시

| 관련도 범위 | 배지 색상 | 의미 |
|:---:|------|------|
| ≥ 50 | 빨간색 (`bg-red-100`) | 이 학생에게 특히 시급한 경로 |
| 25 ~ 49 | 주황색 (`bg-amber-100`) | 보완이 필요한 경로 |
| < 25 | 회색 (`bg-gray-100`) | 현재 큰 문제는 아닌 경로 |

### 5.4 관련 요인 T점수 표시


## 6. 부정적 β(Negative Beta)란?

### 6.1 정의

SEM(구조방정식 모델) 분석에서 매개경로의 **간접효과(a×b)**가 **음수**인 경우입니다.
즉, X가 Z를 거쳐 Y에 도달할 때 Y를 **오히려 낮추는** 역방향 경로입니다.

### 6.2 발생 메커니즘

```
일반 매개경로:  X↑ → Z↑ → Y↑  (간접효과 β > 0)
부정적 경로:    X↑ → Z↓ → Y↓  (간접효과 β < 0)
                 또는 X↑ → Z↑ → Y↓
```

예: `부모공부부담↑ → 부모학업지지↓ → 성적만족도↓`

부모가 공부 부담을 많이 느끼면 → 학업 지지 행동이 감시형으로 변질 → 학생의 성적 만족도 저하

### 6.3 해당 경로 3개

| # | 유형 | 경로 (X → Z → Y) | 간접효과 β | 해석 |
|---|------|------------------|:---:|------|
| 1 | 초등 안전균형형 | 자기정서인식 → 의미감 → 성적만족도 | **-0.033** | 과도한 자기 인식이 부담으로 전환되어 성적만족도를 미세하게 저하 |
| 2 | 중등 무기력형 | 부모공부부담 → 부모학업지지 → 성적만족도 | **-0.077** | 부모의 부담 인식이 학업지지를 약화시켜 만족도 저하 |
| 3 | 중등 무기력형 | 부모공부부담 → 자기효능감 → 성적만족도 | **-0.047** | 부모의 부담이 학생의 자기효능감을 저해하여 만족도 하락 |

### 6.4 시스템 처리

| 항목 | 처리 방식 |
|------|-----------|
| `intervention.beta` | **절대값 \|β\|** 로 저장 (양수). 랭커의 betaBoost 계산에 사용 |
| `intervention.interpretation` | **⚠ 주의:** 접두어 + 원래 음수 값 명시 (β=-0.033 등) |
| 랭킹 | \|β\|가 betaBoost에 반영되므로, 부정적 경로도 관련도가 높게 나올 수 있음 |
| 교사 안내 | "이 경로는 주의가 필요합니다" — 이 방향으로 개입하면 역효과가 날 수 있으므로, **원인(X) 완화가 우선**이라는 전략을 제공 |

## 10. 지식그래프(KG)와 Intervention의 관계

시스템은 **2개 레이어**로 구성됩니다:

### 10.1 KG 레이어 (`knowledgeGraph.ts`)

연구 논문의 SEM 분석 결과를 **모델 단위**로 저장합니다.

- **매개경로 엣지 13개**: Z→Y 모델 기준 그룹화. 하나의 엣지에 여러 X가 포함될 수 있음
  - 예: `자기효능감/자아존중감/공부부담/유능성 → 성장마인드셋 → 성적만족도` = 1개 엣지
- **조절효과 엣지 25개**: X × Z → Y 각각 별도 엣지
- 통계 계수(β, p, CI) 원본 보존

### 10.2 Intervention 레이어 (`lpaProfiles.ts`)

KG 모델을 **X변수 별로 분해**하여 랭킹 가능한 단위로 만듭니다.

- KG의 1개 매개경로 모델 → X 변수 수만큼 intervention으로 분해
  - 예: 위 모델 → `자기효능감 → 성장마인드셋 → 성적만족도`, `자아존중감 → 성장마인드셋 → 성적만족도` 등 4개 intervention
- 랭커가 X 변수의 T점수를 매칭하여 needScore 계산 가능

```
KG 원본 (HF 연구)     KG 엣지 (모델 그룹화)     Intervention (X별 분해)
   ~58개 매개경로  →      13개 엣지         →       57개 intervention
   ~25개 조절효과  →      25개 엣지         →        (조절은 1:1 대응)
```

## 11. 경로 해석(Interpretation) 작성 원칙

| 원칙 | 설명 | 예시 |
|------|------|------|
| 교사 가독성 | 전문 용어 최소화, 행동 중심 서술 | "할 수 있다는 신념이 노력으로 발전한다는 신념으로 전환되어야 지속적 성취" |
| β 값 명시 | 검증된 경로는 β 계수를 괄호 안에 포함 | "(간접효과 β=0.122)" |
| 부정 경로 표시 | ⚠ 주의 접두어 + 원래 음수 값 | "⚠ 주의: ... (간접효과 β=-0.077)" |
| 방향성 안내 | 개입 시 순서나 주의점을 해석에 포함 | "부모 부담 인식 완화가 선행 필요" |

## 12. 구체적 실행 전략(Strategies) 작성 원칙

각 경로에 **3개의 구체적 실행 전략**이 포함됩니다.

| 원칙 | 설명 |
|------|------|
| 즉시 실행 가능 | 교사가 내일 바로 시도할 수 있는 행동 |
| 단계적 구조 | 가능한 1단계 → 2단계 → 3단계 순서로 구성 |
| 유형 맞춤 | 같은 Z→Y 모델이라도 유형에 따라 전략 차별화 (예: 정서조절취약형은 "정서 안정 후 성장 관점 도입") |
| 교사/부모/학생 역할 구분 | 교사정서지지 경로는 교사 행동 중심, 부모 경로는 부모 상담 안내 포함 |
