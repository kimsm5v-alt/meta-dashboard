# META 학습종합검사 구조

## 1. 측정 영역 개요

META 학습종합검사는 **5개 대분류(Depth 1)** 영역으로 구성됩니다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    META 학습종합검사 구조                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌──────────┐  ┌──────────┐  ┌──────────┐                     │
│   │ 자아 강점 │  │학습 디딤돌│  │학습 걸림돌│                     │
│   │ (정적요인)│  │ (정적요인)│  │ (부적요인)│                     │
│   └──────────┘  └──────────┘  └──────────┘                     │
│                                                                 │
│   ┌─────────────────┐  ┌─────────────────┐                     │
│   │ 긍정적 공부마음  │  │ 부정적 공부마음  │                     │
│   │   (정적요인)     │  │   (부적요인)     │                     │
│   └─────────────────┘  └─────────────────┘                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

## 2. 영역별 상세 구조

### 2.1 자아 강점 (Depth 1) - 정적요인

자신의 가치와 능력에 대한 긍정적인 인식과 대인관계능력을 파악합니다.

| Depth 2 (중분류) | Depth 3 (변인명) | 정의 |
|------------------|------------------|------|
| **긍정적 자아** | 자아존중감 | 자신의 가치를 인정하고 소중히 여기는 정도 |
| | 자기효능감 | 어떤 일이든 잘 할 수 있다는 자신감 |
| | 성장마인드셋 | 노력을 통해 성장할 수 있다는 믿음 |
| **대인관계능력** | 자기정서인식 | 자신의 감정을 인식하는 능력 |
| | 자기정서조절 | 자신의 감정을 조절하는 능력 |
| | 타인정서인식 | 타인의 감정을 인식하는 능력 |
| | 타인공감능력 | 타인의 감정에 공감하는 능력 |

### 2.2 학습 디딤돌 (Depth 1) - 정적요인

학습에 도움이 되는 메타인지, 학습기술, 지지적 관계를 확인합니다.

| Depth 2 (중분류) | Depth 3 (변인명) | 정의 |
|------------------|------------------|------|
| **메타인지** | 계획능력 | 학습 계획을 세우는 능력 |
| | 점검능력 | 학습 과정을 점검하는 능력 |
| | 조절능력 | 학습 전략을 조절하는 능력 |
| **학습기술** | 인지전략 | 효과적인 학습 전략 활용 |
| | 시간관리 | 학습 시간을 효율적으로 관리하는 능력 |
| **지지적 관계** | 부모지지 | 부모로부터 받는 학습 지지 |
| | 교사지지 | 교사로부터 받는 학습 지지 |
| | 친구지지 | 친구로부터 받는 학습 지지 |

### 2.3 학습 걸림돌 (Depth 1) - 부적요인

학습을 어렵게 하는 방해요인들을 파악합니다.

| Depth 2 (중분류) | Depth 3 (변인명) | 정의 |
|------------------|------------------|------|
| **학업스트레스** | 학업부담 | 학업에 대한 심리적 부담 |
| | 성적불안 | 성적에 대한 불안감 |
| **학업관계스트레스** | 부모압력 | 부모로부터 받는 성적 압력 |
| | 교사압력 | 교사로부터 받는 성적 압력 |
| | 또래비교 | 또래와의 비교로 인한 스트레스 |
| **학습방해물** | 게임 과몰입 | 인터넷 게임으로 인한 학습 방해 |
| | 스마트폰 과의존 | 스마트폰 사용으로 인한 학습 방해 |

### 2.4 긍정적 공부마음 (Depth 1) - 정적요인

공부하고 싶은 마음, 심리적 욕구를 알아봅니다.

| Depth 2 (중분류) | Depth 3 (변인명) | 정의 |
|------------------|------------------|------|
| **학업열의** | 활력 | 학습에 대한 에너지와 열정 |
| | 헌신 | 학습에 대한 몰입과 집중 |
| | 몰두 | 학습에 완전히 빠져드는 상태 |
| **성장력** | 내재동기 | 학습 자체에 대한 흥미 |
| | 학습가치 | 학습의 중요성에 대한 인식 |

### 2.5 부정적 공부마음 (Depth 1) - 부적요인

공부하기 싫은 마음, 심리적으로 불편한 부분을 알아봅니다.

| Depth 2 (중분류) | Depth 3 (변인명) | 정의 |
|------------------|------------------|------|
| **학업소진** | 고갈 | 학업으로 인한 정서적 탈진 |
| | 냉담 | 학업에 대한 무관심과 거리두기 |
| | 무능감 | 학업에서의 자신감 상실 |

## 3. 전체 구조 요약

### 3.1 계층 구조 (Depth별)

```
Depth 1 (대분류)     Depth 2 (중분류)      Depth 3 (변인명)
─────────────────────────────────────────────────────────
자아 강점           ├─ 긍정적 자아        ├─ 자아존중감
(정적)              │                     ├─ 자기효능감
                    │                     └─ 성장마인드셋
                    └─ 대인관계능력       ├─ 자기정서인식
                                          ├─ 자기정서조절
                                          ├─ 타인정서인식
                                          └─ 타인공감능력
─────────────────────────────────────────────────────────
학습 디딤돌         ├─ 메타인지           ├─ 계획능력
(정적)              │                     ├─ 점검능력
                    │                     └─ 조절능력
                    ├─ 학습기술           ├─ 인지전략
                    │                     └─ 시간관리
                    └─ 지지적 관계        ├─ 부모지지
                                          ├─ 교사지지
                                          └─ 친구지지
─────────────────────────────────────────────────────────
학습 걸림돌         ├─ 학업스트레스       ├─ 학업부담
(부적)              │                     └─ 성적불안
                    ├─ 학업관계스트레스   ├─ 부모압력
                    │                     ├─ 교사압력
                    │                     └─ 또래비교
                    └─ 학습방해물         ├─ 게임 과몰입
                                          └─ 스마트폰 과의존
─────────────────────────────────────────────────────────
긍정적 공부마음     ├─ 학업열의           ├─ 활력
(정적)              │                     ├─ 헌신
                    │                     └─ 몰두
                    └─ 성장력             ├─ 내재동기
                                          └─ 학습가치
─────────────────────────────────────────────────────────
부정적 공부마음     └─ 학업소진           ├─ 고갈
(부적)                                    ├─ 냉담
                                          └─ 무능감
```

### 3.2 요인 수 요약

| 구분 | 개수 |
|------|------|
| Depth 1 (대분류) | 5개 |
| Depth 2 (중분류) | 11개 |
| Depth 3 (변인명) | 38개 |
| 총 문항 수 | 124문항 |

### 3.3 정적/부적 요인 구분

| 요인 유형 | 해당 영역 | 해석 방향 |
|-----------|-----------|-----------|
| **정적요인** | 자아 강점, 학습 디딤돌, 긍정적 공부마음 | 높을수록 긍정적 |
| **부적요인** | 학습 걸림돌, 부정적 공부마음 | 높을수록 부정적 |

## 4. API 데이터 모델링 참고

### 4.1 영역 구조 (TypeScript)

```typescript
interface MeasurementArea {
  depth: 1 | 2 | 3;
  code: string;           // 예: "SELF_STRENGTH"
  name: string;           // 예: "자아 강점"
  fullName: string;       // 예: "자아 강점>긍정적 자아>자아존중감"
  factorType: 'positive' | 'negative';  // 정적/부적
  parentCode?: string;    // 상위 영역 코드
}
```

### 4.2 영역 코드 체계 (권장)

```
Depth 1: SELF_STRENGTH, LEARNING_STEPPING, LEARNING_OBSTACLE, 
         POSITIVE_MIND, NEGATIVE_MIND

Depth 2: POSITIVE_SELF, INTERPERSONAL, METACOGNITION, 
         LEARNING_SKILL, SUPPORTIVE_REL, ACADEMIC_ENTHUSIASM,
         GROWTH_POWER, ACADEMIC_STRESS, RELATION_STRESS,
         LEARNING_DISTURBANCE, ACADEMIC_BURNOUT

Depth 3: SELF_ESTEEM, SELF_EFFICACY, GROWTH_MINDSET, ...
```
