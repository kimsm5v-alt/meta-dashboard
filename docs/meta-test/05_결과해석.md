## 3. API 해석 스크립트 데이터 구조

### 3.1 TypeScript 인터페이스

```typescript
interface InterpretationScript {
  sectionFullName: string;    // "자아 강점>긍정적 자아"
  sectionName: string;        // "긍정적 자아"
  level: '매우낮음' | '낮음' | '보통' | '높음' | '매우높음';
  tScoreLower: number;        // T점수 하한
  tScoreUpper: number;        // T점수 상한
  colorTeacher: string;       // 교사용 색상
  colorStudent: string;       // 학생용 색상
  script: string;             // 해석 스크립트
}
```

### 3.2 해석 스크립트 조회 로직

```typescript
function getInterpretationScript(
  sectionName: string,
  tScore: number,
  scripts: InterpretationScript[]
): InterpretationScript | null {
  return scripts.find(s => 
    s.sectionName === sectionName &&
    tScore >= s.tScoreLower &&
    tScore <= s.tScoreUpper
  ) || null;
}
```

### 4.2 해석 순환 구조

1. **긍정적 자아 & 지지적 관계** → 긍정적 공부마음 ↑
2. **긍정적 공부마음** → 메타인지 & 학습기술 ↑
3. **메타인지 & 학습기술** → 긍정적 자아 강화 (선순환)

1. **학업스트레스 & 학업관계스트레스** → 부정적 공부마음 ↑
2. **부정적 공부마음** → 자기정서조절 ↓
3. **자기정서조절 저하** → 학업스트레스 ↑ (악순환)

## 5. 코칭 방향 제시

### 5.1 영역별 개입 우선순위

| 우선순위 | 조건 | 개입 방향 |
|----------|------|-----------|
| 1순위 | 학업소진 매우 높음 | 심리적 지원 및 휴식 제공 |
| 2순위 | 자기정서조절 매우 낮음 | 정서 조절 훈련 |
| 3순위 | 메타인지 낮음 | 학습 전략 코칭 |
| 4순위 | 지지적 관계 낮음 | 관계 지원 강화 |

### 5.2 강점 기반 코칭

- 높음/매우 높음 영역을 **강점**으로 활용
- 강점을 기반으로 약점 영역 개선 유도
- 작은 성공 경험 축적 → 자기효능감 향상
