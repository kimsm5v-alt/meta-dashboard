# 3단계 대시보드 설계 문서

## 개요

META 학습심리정서검사 결과를 분석하고 시각화하는 3단계 대시보드 설계

---

## Level 1: 교사 전체 반 대시보드

> **목적**: 담당하는 모든 반의 현황을 한눈에 비교/파악

### 1.1 헤더 정보
```
┌─────────────────────────────────────────────────────────┐
│ 김교사님의 학급 현황                                      │
│ 담당 학급: 3개 반 | 총 학생: 87명 | 검사 완료: 82명 (94%)  │
└─────────────────────────────────────────────────────────┘
```

### 1.2 반별 카드 리스트
```
┌─────────────────────┐  ┌─────────────────────┐  ┌─────────────────────┐
│ 6학년 2반           │  │ 6학년 3반           │  │ 6학년 5반           │
│ 학생 29명           │  │ 학생 30명           │  │ 학생 28명           │
│ 검사완료 28명 (97%) │  │ 검사완료 27명 (90%) │  │ 검사완료 27명 (96%) │
├─────────────────────┤  ├─────────────────────┤  ├─────────────────────┤
│ 유형 분포:          │  │ 유형 분포:          │  │ 유형 분포:          │
│ 🟠 자원소진 8명     │  │ 🟠 자원소진 12명    │  │ 🟠 자원소진 5명     │
│ 🔵 안전균형 12명    │  │ 🔵 안전균형 9명     │  │ 🔵 안전균형 14명    │
│ 🔷 몰입풍부 8명     │  │ 🔷 몰입풍부 6명     │  │ 🔷 몰입풍부 8명     │
├─────────────────────┤  ├─────────────────────┤  ├─────────────────────┤
│ ⚠️ 관심 필요: 3명   │  │ ⚠️ 관심 필요: 5명   │  │ ⚠️ 관심 필요: 2명   │
│ [상세보기]          │  │ [상세보기]          │  │ [상세보기]          │
└─────────────────────┘  └─────────────────────┘  └─────────────────────┘
```

### 1.3 전체 반 비교 차트
- 가로 막대 차트: 반별 유형 분포 비교
- 색상: 🟠 자원소진형 / 🔵 안전균형형 / 🔷 몰입자원풍부형

### 1.4 전체 요약 통계
- 유형별 총계
- 관심 필요 학생 (학업소진, 스마트폰 의존, 부모 압박)

### 1.5 차수별 비교 (2차 검사 완료 시)
- 유형 변화 학생 수
- 개선/악화 비율
- 반별 개선율

---

## Level 2: 특정 반 대시보드

> **목적**: 해당 반의 전체 현황 + 학생 목록 + 반 특성 파악

### 2.1 헤더 정보
```
┌─────────────────────────────────────────────────────────┐
│ 🏫 6학년 2반 대시보드                                    │
│ 담임: 김교사 | 학생: 29명 | 검사완료: 28명 (97%)         │
│ 1차 검사: 2026.03.15 | 2차 검사: 2026.09.20             │
│                                                         │
│ [차수 선택] ◉ 1차  ○ 2차  ○ 1차↔2차 비교               │
└─────────────────────────────────────────────────────────┘
```

### 2.2 유형별 학생 분포도
- 파이/도넛 차트
- 유형명 + 학생수 + 퍼센트
- 클릭 시 해당 유형 학생만 필터링
- 미실시 학생 포함

### 2.3 차수별 비교 (2단 레이아웃)
```
┌──────────────────────────┬──────────────────────────────┐
│ 1차 검사 (2026.03.15)    │ 2차 검사 (2026.09.20)        │
├──────────────────────────┼──────────────────────────────┤
│ 🟠 자원소진: 10명 (36%)  │ 🟠 자원소진: 8명 (29%) ▼    │
│ 🔵 안전균형: 11명 (39%)  │ 🔵 안전균형: 12명 (43%) ▲   │
│ 🔷 몰입풍부: 7명 (25%)   │ 🔷 몰입풍부: 8명 (29%) ▲    │
├──────────────────────────┴──────────────────────────────┤
│ 📈 변화 요약: 유형 변화 4명 (개선 3명 / 악화 1명)        │
└─────────────────────────────────────────────────────────┘
```

### 2.4 반 전체 요인 프로필
- 5대 영역 평균 T점수 (바 차트)
- 전체 평균(50) 대비 비교
- AI 학급 특성 요약

### 2.5 관심 필요 학생 알림
- 학업소진 높음
- 스마트폰 의존 위험
- 부모 압박 높음

### 2.6 학생 목록 테이블
| 번호 | 이름 | 1차 유형 | 2차 유형 | 주요 키워드 | 확신도 |
|------|------|----------|----------|-------------|--------|

- 필터: 유형별, 관심 필요, 변화 상태
- 정렬: 번호, 이름, 유형, 확신도
- 검색: 이름/번호
- 클릭 시 L3 이동

---

## Level 3: 특정 학생 대시보드

> **목적**: 개인 진단 결과 상세 + 유형 분석 + 맞춤 코칭 전략

### 3.1 헤더 정보
```
┌─────────────────────────────────────────────────────────┐
│ 👤 7번 학생 상세 진단                                    │
│ 6학년 2반 | 검사일: 1차 2026.03.15, 2차 2026.09.20      │
│                                                         │
│ [차수 선택] ○ 1차  ◉ 2차 (최신)  ○ 1차↔2차 비교        │
│                                                         │
│ [← 반 대시보드로 돌아가기]  [이전 학생] [다음 학생]      │
└─────────────────────────────────────────────────────────┘
```

### 3.2 유형 분류 결과
```
┌─────────────────────────────────────────────────────────┐
│ 🏷️ 학습 유형 분석                                        │
├─────────────────────────────────────────────────────────┤
│   ┌─────────┐                                           │
│   │  🟠     │  자원소진형                               │
│   │  87%    │  (확신도 87%)                             │
│   └─────────┘                                           │
│                                                         │
│   전체 유형 확률:                                        │
│   🟠 자원소진형:     ████████████████░░░░ 87%           │
│   🔵 안전균형형:     ██░░░░░░░░░░░░░░░░░░ 11%           │
│   🔷 몰입자원풍부형: ░░░░░░░░░░░░░░░░░░░░  2%           │
├─────────────────────────────────────────────────────────┤
│ 📝 유형 설명: "학습에 필요한 심리·정서적 자원이..."      │
├─────────────────────────────────────────────────────────┤
│ 🔍 이 학생만의 특이점 (유형 평균 대비):                  │
│ 1. 🔴 무능감 T=68 (유형평균 58.7보다 +9.3 높음)         │
│ 2. 🔴 고갈 T=72 (유형평균 56.2보다 +15.8 높음) ⚠️       │
│ 3. 🔵 수업태도 T=52 (유형평균 41.7보다 +10.3 높음)      │
└─────────────────────────────────────────────────────────┘
```

### 3.3 차수별 변화 (비교 모드)
- 유형 변화 여부
- 확신도 변화
- 주요 요인 변화 (개선/악화)

### 3.4 진단결과 한눈에 보기 (11개 중분류)
- 가로 바 차트
- T점수 20~80 범위
- 평균선(50) 표시
- 차수 비교 시 1차(회색) vs 2차(컬러) 오버레이

### 3.5 AI 총평
- 전반적 상태 요약 (2-3문장)
- 강점 강조
- 주의 필요 영역

### 3.6 코칭 전략 (지식그래프 기반)
```
┌─────────────────────────────────────────────────────────┐
│ 🎯 맞춤 코칭 전략 (지식그래프 기반)                       │
├─────────────────────────────────────────────────────────┤
│ [추천 경로 1] ⭐ 최우선                                  │
│ ┌─────────────────────────────────────────────────────┐ │
│ │ 자기효능감 ──→ 자아존중감 ──→ 학업성취도            │ │
│ │              (매개)           (결과)                 │ │
│ └─────────────────────────────────────────────────────┘ │
│ 효과 유형: 긍정완충 (동시 개입 시 효과 감소)             │
│ 💡 전략: 순차적 개입 - 효능감 먼저, 자존감은 나중에      │
│                                                         │
│ 구체적 실행:                                             │
│ • 1단계: 작은 성공 경험 제공 (쉬운 과제 → 성취감)       │
│ • 2단계: 과정 칭찬 ("3번이나 다시 시도했구나")          │
│ • 3단계: 이후 정서적 지지 ("넌 괜찮은 학생이야")        │
└─────────────────────────────────────────────────────────┘
```

### 3.7 데이터 해석 도우미 (AI 챗)
- 추천 질문 프리셋
- 자유 질문 입력
- 스트리밍 응답

### 3.8 빠른 메뉴
- AI 상담실에서 분석하기
- 생활기록부 문구 생성
- 상담 기록 작성

```

    │                        ├── 학생 행 클릭 ──→ L3 (특정 학생)
    │                        │                        │
    │                        │                        ├── AI 상담실

```
