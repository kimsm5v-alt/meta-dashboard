## 🔑 핵심 요약

### 검사 구조
```
5개 대분류 → 11개 중분류 → 38개 변인 → 124문항
```

### 점수 체계
- **T점수**: 평균 50, 표준편차 10
- **등급**: 매우낮음(~29) / 낮음(30~39) / 보통(40~59) / 높음(60~69) / 매우높음(70~)

### LPA 유형 (초등)
| 유형 | 비율 | 특징 |
|------|------|------|
| 자원소진형 | 30.6% | 자기효능감·정서조절 낮음 |
| 안전균형형 | 35.5% | 전반적 균형 |
| 몰입자원풍부형 | 34.0% | 내적 동기 높음 |

### 신뢰도 지표
- 사회적 바람직성: T≥65 → 주의
- 반응 일관성: T≥65 → 주의
- 연속동일반응: 10개 이상 → 주의

## 🛠️ 개발 참고

### 필수 데이터 파일
1. 학교급별 기준 데이터 (평균, 표준편차)
2. LPA 프로파일 데이터 (38개 변인 평균)
3. 해석 스크립트 데이터 (등급별 문구)
4. 문항 데이터 (문항 내용, 순서)

### 주요 계산 로직
1. 원점수 → T점수 변환
2. T점수 → 등급 판정
3. 38개 T점수 → LPA 유형 분류
4. 신뢰도 지표 산출
